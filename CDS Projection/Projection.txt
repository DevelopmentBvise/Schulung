To make a reference to an element of the SELECT list in the ON condition, the field name can be prefixed with $projection instead of the source data source. In this case, an alternative element name defined using AS can be specified instead of the field name. If a path expression of the SELECT list is used when the prefix $projection is used in the ON condition, the current association cannot itself be used in the SELECT list, to avoid invalid join expressions.
